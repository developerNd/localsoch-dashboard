Common Issues

  • Backend not restarted: Changes won't take effect until restart
  • File upload service disabled: Check if Strapi upload plugin is enabled
  • File size too large: Check file size limits
  • File type not allowed: Check allowed file types
  • Storage permissions: Check if upload directory is writable
